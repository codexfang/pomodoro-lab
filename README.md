# Pomodoro Lab

A modern productivity timer and focus analytics system that tracks your work sessions, visualizes productivity patterns, and stays in the zone with the Pomodoro technique.

## Features

- **Pomodoro Timer** — Customizable focus, short break, and long break sessions with visual progress ring
- **Task Tagging** — Assign task names to each session and track where your time goes
- **Session History** — Persistent log of every completed session with timestamps and durations
- **Analytics Dashboard** — Total focus time, daily/weekly breakdowns, task distribution charts, and productivity insights
- **Notifications** — Browser notifications and dynamic document title updates when sessions end
- **Theme Support** — Light, dark, and system-aware theme switching
- **Sound Alerts** — Optional sound notifications for session transitions
- **Fully Offline** — All data stored in localStorage, no backend required

## Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool and dev server
- **Zustand** — State management
- **Recharts** — Data visualization
- **date-fns** — Date utilities
- **Browser Notifications API** — Session completion alerts
